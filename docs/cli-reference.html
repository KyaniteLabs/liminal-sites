💻 CLI Reference

Command-line interface for Liminal

🚀 Commands

liminal tui

Launch the interactive Bubble Tea operator TUI via the GLM-backed bridge bootstrap script.

$ liminal tui

# Or use the shorthand alias:
$ lim tui

Inside the TUI, you can type naturally:

  • "Fix the validation" — Agent mode
  • "What's the status?" — Command mode
  • "Tell me about p5.js" — Chat mode

liminal --prompt "..."

Generate art directly from the command line.

$ liminal --prompt "Create a calming blue particle system"
$ liminal --prompt "Generate a shader with Perlin noise" --domain glsl

liminal serve

Start the preview server for browser-based viewing.

$ liminal serve
# Server starts on http://localhost:3000

Bubble Tea TUI (Go)

Next-generation operator-grade terminal UI. The recommended entrypoint is pnpm tui, which builds the repo, starts the TS bridge, and launches Bubble Tea.

$ cd bubbletea && go run .

# Or with a custom bridge URL:
$ LIMINAL_BRIDGE_URL=http://localhost:3000 go run .

⚙️ Configuration

Environment Variables

Copy .env.example to .env and configure:

Variable Description Default
LIMINAL_LLM_PROVIDER LLM provider (lmstudio, ollama, openai) lmstudio
LIMINAL_LLM_BASE_URL Base URL for LLM API http://localhost:1234/v1
LIMINAL_LLM_MODEL Model name to use qwen2.5-coder-7b-instruct
OPENAI_API_KEY OpenAI API key (if using OpenAI) -
LIMINAL_ALLOW_LOCALHOST_LLM Allow localhost connections true

Example .env

# LM Studio (default)
LIMINAL_LLM_PROVIDER=lmstudio
LIMINAL_LLM_BASE_URL=http://localhost:1234/v1
LIMINAL_LLM_MODEL=qwen2.5-coder-7b-instruct

# Or Ollama
LIMINAL_LLM_PROVIDER=ollama
LIMINAL_LLM_BASE_URL=http://localhost:11434
LIMINAL_LLM_MODEL=codellama:7b

# Or OpenAI
LIMINAL_LLM_PROVIDER=openai
LIMINAL_LLM_MODEL=gpt-4
OPENAI_API_KEY=sk-...

🎮 TUI Commands

When inside the TUI, you can use explicit commands with / prefix:

Command Description
/statusShow harness status
/tasksList pending harness tasks
/run <task-id>Execute a task (e.g., /run M1)
/preview <file>Preview file (auto-routes)
/play <audio>Play audio file
/stopStop audio playback
/browserOpen browser preview
/helpShow help
/clearClear screen
/exitQuit TUI
