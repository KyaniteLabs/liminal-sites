🚀 Quick Start

# Install dependencies
pnpm install

# Start the TUI with natural language interface
liminal tui

# Or use the alias
lim tui

# Inside the TUI, just type naturally:
# → "Fix the Tone.js validation"
# → "What's the status?"
# → "Tell me about p5.js"
