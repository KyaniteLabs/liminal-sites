Architecture Overview

48 subsystems organized into functional layers.

Core Loop

src/core/, src/llm/, src/config/

RalphLoop engine, LLM client with provider adapters, configuration loading.

Generators

src/generators/

11 domain generators with TierBasedGenerator base class.

Meta-Harness

src/harness/, src/guardrails/

Failure logging, pattern detection, self-improvement, M1-M18 guardrails.

Intelligence

src/brain/, src/compost/, src/evolution/

Artistic knowledge, Compost Mill, MAP-Elites, novelty archive.

Audio & Music

src/audio/, src/music/

Audio analysis, pitch detection, Euclidean rhythms, Markov chains.

Aesthetics

src/aesthetic/, src/calibration/

Color theory, design tiers, multi-critic evaluation pipeline.

TUI

src/tui/, src/tui-bridge/, src/chat/

Bubble Tea bridge, interactive chat sessions.

Security & Infra

src/security/, src/sandbox/, src/render/

SSRF protection, rate limiting, rendering pipeline.

Task Ledger

src/ledger/, src/fs/

Self-hosting task corpus, runner, verifier, LiminalFS substrate.

Collaboration

src/collab/, src/swarm/, src/routing/

Multi-agent board, swarm generation, dual-model routing.

Learning

src/intuition/, src/learning/, src/embeddings/

Thompson Sampling, domain prototypes, local SBERT embeddings.

Plugins & Quality

src/plugins/, src/quality/, src/fix/

Plugin system, quality metrics, auto-fix with LLM.

Quick Start

# Install
pnpm install

# Configure (first time)
liminal --configure

# Generate
liminal --prompt "Create a calming blue particle system"

# Chat-driven creative session
liminal chat

# Terminal UI (Bubble Tea — requires Go >= 1.21)
pnpm run tui
