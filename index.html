How It Works

The Ralph-Wiggum Loop: same prompt, changing world, self-critique, emergent quality.

Prompt

Same prompt every iteration. You set the creative direction — Liminal handles the rest.

Iterate

Context changes each round — previous work, scores, compost seeds. The world evolves.

Curate

Quality gates filter the best. Gallery saves iterations. You pick what's worth keeping.

// The core loop — deceptively simple while iterations < max: code = generate(prompt, context) // LLM creates score = evaluate(code) // quality gate ≥ 0.7 context.accumulate(code, score) // world changes if promise_detected: break // done improving
CAPABILITIES

Beyond One-Shot Generation

Swarm Generation

7 creative personas run in parallel via Ollama — Architect, Synthesizer, Explorer, Muse, Distiller. Competitive, hybrid, ring, and mesh modes with LLM-powered voting on the final round.

Deep Collaboration

Multi-model specialist pipeline: Creator + Visionary diverge, 3 critics analyze, Integrator + Refiner synthesize. Converges automatically across local and cloud models.

Compost Mill

Feed it files, directories, or previous outputs. Extracts, shreds, collides fragments across domains, scores quality, and evolves seeds into the generation loop.

Live Music Coding

Generate Strudel (TidalCycles) and Hydra (audio-reactive visuals) code. Music-to-visual bridge extracts BPM and FFT to drive visual generation.

QUICK START

Up and Running in 60 Seconds

# Install npm install

# Configure an LLM backend liminal --configure

# Generate liminal --prompt "A calming blue particle system"
# Or use Ollama locally export LIMINAL_LLM_PROVIDER=ollama
export LIMINAL_LLM_MODEL=llama3.2
liminal --prompt "Fractal growth pattern" --use-swarm
# Feed creative material into the compost mill liminal compost add ./my-sketches/
liminal compost digest
# Seeds now cross-pollinate into every generation

Compost DNA Injection

Promoted seeds register as DNA in GeneratorRegistry. When a prompt matches, core patterns inject into context.

MAP-Elites Diversity

When coverage drops below 30%, the system injects diversity hints to explore under-explored behavior regions.

Swarm → Archive

Semantic Few-Shot

Archive examples ranked by keyword overlap with current prompt, budgeted to 2000 chars for context efficiency.

Auto-Compost Cycle

Quality outputs auto-feed compost heap. When full, digest triggers automatically. Seeds inject into next loop.

Cross-Run Persistence

MAP-Elites, aesthetic model, and routing data persist to disk. Each run builds on all previous runs.
