p5.js

Flowing Particle Field

Prompt: "calming blue particles flowing like water"

// P5.js generative art with Perlin noise flow field
function setup() {
  createCanvas(800, 600);
  background(5, 10, 30);
}

function draw() {
  for (let x = 0; x < width; x += 20) {
    for (let y = 0; y < height; y += 20) {
      let n = noise(x * 0.01, y * 0.01, frameCount * 0.01);
      let angle = n * TWO_PI * 2;
      let px = x + cos(angle) * 15;
      let py = y + sin(angle) * 15;

      stroke(100 + n * 100, 150 + n * 80, 255, 150);
      line(x, y, px, py);
    }
  }
}

Visual Output

A flowing field of blue particles that move organically using Perlin noise. The particles create wave-like patterns that mimic water flowing across the canvas.

GLSL Shader

Raymarched Landscape

Prompt: "alien landscape with fog and glowing orbs"

// GLSL fragment shader with raymarching
uniform vec2 u_resolution;
uniform float u_time;

float sdSphere(vec3 p, float r) {
  return length(p) - r;
}

float map(vec3 p) {
  vec3 q = p;
  q.xz *= mat2(0.8, -0.6, 0.6, 0.8);
  float terrain = p.y + 0.5 * sin(q.x * 2.0) * cos(q.z * 2.0);
  float orbs = 1e9;
  for (int i = 0; i < 5; i++) {
    vec3 orbPos = vec3(
      sin(u_time + float(i) * 1.5) * 3.0,
      1.5 + sin(u_time * 2.0 + float(i)),
      cos(u_time + float(i) * 1.3) * 3.0
    );
    orbs = min(orbs, sdSphere(p - orbPos, 0.3));
  }
  return min(terrain, orbs);
}

Visual Output

An alien terrain with undulating hills and 5 glowing orbs that float above the surface. Fog creates depth and the orbs pulse with an ethereal glow.
